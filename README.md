# React Orders Starter

A simple orders management application built with React, TypeScript, and Vite.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Pages

- **Orders List** (`/orders`) - Browse, filter, and search orders with pagination
- **Order Details** (`/orders/:orderId`) - View detailed information about a specific order

## Tech Stack

React 18, TypeScript, Vite, React Router, React Query, Tailwind CSS
